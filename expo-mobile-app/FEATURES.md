# Características de Energy Flow Mobile

## 🎯 Funcionalidades Principales

### 📊 Dashboard en Tiempo Real
- **Métricas en vivo**: Potencia, energía, voltaje y corriente actualizados cada 20 segundos
- **Gráficas interactivas**: Barras, líneas y circulares para visualización de datos
- **Análisis por fases**: Detalles específicos de las fases eléctricas A, B y C
- **Indicador de conexión**: Estado en tiempo real de la conectividad con el sistema
- **Impacto ambiental**: Cálculo automático de emisiones de CO₂

### 🗺️ Mapa Interactivo del Campus
- **Vista geográfica**: Mapa completo del campus universitario
- **Marcadores de edificios**: Identificación visual de cada edificio monitoreado
- **Estados visuales**: Colores diferenciados según el estado operativo
- **Información rápida**: Detalles al tocar cada marcador
- **Estadísticas generales**: Resumen del consumo total del campus

### 🏢 Gestión de Edificios
- **Lista completa**: Todos los edificios monitoreados
- **Búsqueda avanzada**: Filtros por nombre, ubicación y estado
- **Métricas individuales**: Consumo específico por edificio
- **Estados operativos**: Activo, inactivo, mantenimiento
- **Información detallada**: Ubicación, pisos, eficiencia energética

### ⚙️ Configuración y Personalización
- **Perfil de usuario**: Información personal y preferencias
- **Notificaciones**: Configuración de alertas y avisos
- **Tema visual**: Modo claro/oscuro
- **Actualización automática**: Configuración de intervalos
- **Exportación de datos**: Descarga en formatos CSV y PDF

## 🎯 Métricas Monitoreadas

### ⚡ Parámetros Eléctricos
- **Corrientes RMS**: Fases A, B, C (Amperes)
- **Voltajes RMS**: Fases A, B, C (Volts)
- **Voltajes línea a línea**: AB, BC, CA (Volts)
- **Potencias promedio**: Fases A, B, C (Watts)
- **Energía acumulada**: Fases A, B, C (kWh)

### 📊 Métricas Calculadas
- **Potencia total**: Suma de todas las fases
- **Energía total**: Consumo acumulado
- **Voltaje promedio**: Media de las tres fases
- **Corriente promedio**: Media de las tres fases
- **Emisiones CO₂**: Cálculo basado en factor de emisión

### 🏢 Datos por Edificio
- **Estado operativo**: Activo, inactivo, mantenimiento
- **Eficiencia energética**: Porcentaje de eficiencia
- **Ubicación**: Campus y dirección específica
- **Características**: Número de pisos, área, tipo
- **Historial**: Tendencias y patrones de consumo
