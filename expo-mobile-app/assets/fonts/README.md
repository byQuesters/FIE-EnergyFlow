# Fonts

This directory contains custom fonts for the Energy Flow mobile application.

## System Fonts
The application primarily uses system fonts for optimal performance and native feel:
- iOS: San Francisco
- Android: Roboto

## Custom Fonts (if needed)
Place any custom .ttf or .otf files here and reference them in the theme configuration.

## Usage
Fonts are configured in `src/constants/theme.ts` and automatically loaded by Expo.
